Every XCode version changes how to view all breakpoints. XCode 3 `⌘ + ALT + B`. XCode 4 `⌘ + 7`.

**In XCode 5:**

`⌘ + 8` to display all breakpoints

`⌘ + A` to select all breakpoints

`DELETE` to remove them _this can take a few seconds_
